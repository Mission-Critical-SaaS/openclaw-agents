## 2026-03-19 — DCAA Compliance Essentials
- Daily time recording required (no weekly reconstruction)
- Electronic dual signatures (employee + supervisor)
- Full audit trail: timestamp, user ID, change reason
- Charge code separation: direct vs. indirect labor
- Role-based access control (Employee, Supervisor, Accountant, Admin)
- SOC 2 certified data storage
